# Calc

Simple Calculator

## Integer Arithmetic

```Shell
$ zuk (1 + 2 + 3 + 4) * 5
```

## Decimal Arithmetic

```Shell
$ zuk 2900401.23365 * 6316714975174469.1697205940
```
